# Fortune Teller
This webapp provides a random fortune to user and lets users send their friend's fortune by SMS using Twilio's Messaging API.

# Tech Stack
- Python, HTML, CSS, Flask, Bootstrap, Flask-WTF

# API Used
- Twilio Messaging API.
